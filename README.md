# Data Quality & Analytics Portfolio

A collection of data engineering and analytics projects built with Python, SQL, BigQuery, and Google Cloud Platform.

## Projects

### 01 - HR Data Quality Monitoring System
A simulated enterprise data quality monitoring system built on a messy HR dataset. Designed a backend SQL-based validation framework that detects duplicates, enforces rule-based data validation, logs audit results, and measures before/after cleanup impact. Includes a staging table, validation rules catalog, row-level validation results, duplicate detection views, a cleaned employee table, and a flagged-for-review issue queue. Power BI serves as the reporting layer.

**Tech:** SQL, Power BI, Data Governance, Audit Tracking

[View Project](01_HR_Data_Quality_Monitoring/)

### 02 - Retail Stock Market Analyzer
Full end-to-end data pipeline analyzing 5 years of historical stock data across Target, Walmart, Costco, Kroger, and Ulta Beauty. Includes SQL analytics, feature engineering, and a machine learning model built to predict daily price direction.

**Tech:** Python, BigQuery, GCP, scikit-learn, Jupyter
